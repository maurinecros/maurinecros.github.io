Cinéma le festival

Cinéma le Festival est une refonte fictive d'un site web déjà existant.
C'est le premier site web que nous avons codé lors de ma formation en DUT MMI, en 2020.
Nous avons été en groupes de 5 et avons du coder ce site web en 15 jours.

Le visuel sombre au premier abord a été choisi pour faire ressortir les affiches de films.
La deuxième couleur dominante est le mauve.
Elle donne un côté enfantin au site car ce cinéma est spécialisé dans les films d'animations,
notamment pour les enfants.

Le site web est adapté mobile et desktop, pour une fluidité de l'information quelle que soit le support de l'utilisateur. Les informations sont les mêmes, présentés ergonomiquement.

Codé avec l'interface Visual Studio code, utilisant les langages Html, Css et Java Script.
