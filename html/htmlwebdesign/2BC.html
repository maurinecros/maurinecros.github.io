Bears Cooking

Ce site a été conçu dans le cadre de ma formation en DUT MMI. Nous avons crée ce site one page en groupe de 6 en une semaine.
Ce site a pour but d'aider les jeunes et les étudiants à mieux s'alimenter durant le confinement.

Un site web accessible disponible sur téléphone et sur desktop.
Le site se doit d'être atteignable pour que les utilisateurs trouvent les recettes depuis n'importe quel outil.

Le site propose de nombreuses recettes qui se renouvellent chaque jour.
Faciles, peu chères et le tout fait en 30 minutes top chrono !
Idéales pour les étudiants, ce site leur permettra de manger équilibré tout en se régalant.

Codé avec l'interface Visual Studio code, utilisant les langages Html, Css et Java Script.
