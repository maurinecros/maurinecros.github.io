Class Code

Le projet Class Code est un projet de refonte d'un site web auquel j'ai eu l'occasion de participer. Il a été fait dans le cadre de ma formation en DUT MMI. Class Code est un site éducatif de ressources pour les professeurs dans le codage et l'algorithmie.

Des cours sont disponibles sur le site, de manière organisée.
Les vidéos explicatives sont rangés et sont classées par popularité.

Nous avons rendu le site plus clair, sobre et moderne.
La chartre graphique a été conservée en partie en gardant le bleu et le beige.

Le noir rajoute un dynamisme et un aspect professionnel au site.

Le site devient plus clair, les pages sont classées et conservent toutes la même identité graphique.
Class Code propose de nombreuses rubriques (agenda, carte, parcours & ressources…)

Codé avec l'interface Visual Studio code, utilisant les langages Html, Css et Java Script.
Le design a été fait sur Figma.
